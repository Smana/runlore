ownership** so no two agents ever edit the same file (clean integration merges guaranteed).

(integration owner) merges each workstream branch sequentially, then runs the **full** CI gate
suite once on the integrated tree: SPDX headers, `go build`, `go vet`, `gofmt -l`,
`go test -race`, govulncheck, golangci-lint v2.12.2.

New `.go` files MUST start with `// SPDX-License-Identifier: Apache-2.0` as line 1.
