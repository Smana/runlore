# v0.9.0 execution plan — 2026-07-11

Executes the confirmed findings and roadmap from `2026-07-11-pre-announcement-audit.md` /
`-roadmap.md`. Theme of the release: **the correctness + safety release** — make the advertised
quickstart boot, stop silent alert drops, and make the correlation differentiator (change↔symptom
timelines, log attribution) actually work.

## Method

Parallel implementation in **isolated git worktrees**, partitioned by **exclusive file
ownership** so no two agents ever edit the same file (clean integration merges guaranteed).
Each agent works TDD, runs scoped gates in its worktree, and commits. The controller
(integration owner) merges each workstream branch sequentially, then runs the **full** CI gate
suite once on the integrated tree: SPDX headers, `go build`, `go vet`, `gofmt -l`,
`go test -race`, govulncheck, golangci-lint v2.12.2.

New `.go` files MUST start with `// SPDX-License-Identifier: Apache-2.0` as line 1.
