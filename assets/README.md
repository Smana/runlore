# runlore logo assets

Direction **1a** — geometric badge. Navy `#101f4b`, electric blue `#3b82f6`, teal accent `#14c9a6`.

| File | Use |
|---|---|
| `logo.svg` / `logo.png` | Primary circle badge (light bg `#eff1f5`). 512px PNG. Drop-in replacement for the old `logo.png`. |
| `logo-dark.svg` / `logo-dark.png` | Rounded-square badge on dark `#0b1533`. |
| `logo-mark.svg` / `favicon-256.png` / `favicon-64.png` | Mark only, **transparent** facial cutout — favicons, avatars, any background. |
| `logo-wordmark.svg` | Horizontal `runlore` lockup (needs the Space Grotesk font, with system-sans fallback). |

SVGs are the source of truth — scale losslessly and re-color by editing the `fill` values.
